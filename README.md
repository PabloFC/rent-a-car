# 🚗 Rent a Car

Aplicación web para alquiler de coches.

## 🛠️ Stack Tecnológico

### Frontend

- React
- Tailwind CSS
- Vite

### Backend

- Node.js
- Express
- PostgreSQL
- Prisma

## 🚀 Instalación

### Backend

```bash
cd backend
npm install
cp .env.example .env
# Configurar las variables de entorno en .env
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 📋 Funcionalidades

- ✅ Listado de autos
- ✅ Reserva de autos
- ✅ Panel de administración
- ✅ Gestión de imágenes de autos
- ✅ Pago simulado
- ✅ Autenticación (Usuario/Admin)
